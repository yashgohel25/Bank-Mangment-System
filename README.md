# 🏦 Simple Banking System (Python)

A beginner-friendly banking system simulation built with Python.  
It allows users to create accounts, deposit and withdraw money, view account details, and update account information.  
All data is stored persistently in a JSON file (`data.json`).

---

## ✨ Features
- Create account (validates age ≥ 18 and 4-digit PIN)
- Deposit money (secure with account number + PIN)
- Withdraw money (checks balance before withdrawal)
- View account details
- Update account details (name, email, PIN)
- Persistent storage in `data.json`
